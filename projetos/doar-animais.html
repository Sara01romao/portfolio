Cadastro de Animais para Doação

O projeto de cadastro de animais para doação tem como objetivo praticar a criação de formulários e utilizar a API de localidades do IBGE para obter estados e suas respectivas cidades. Este projeto visa facilitar o processo de doação de animais, permitindo que os usuários registrem animais disponíveis para adoção de maneira organizada e eficiente
